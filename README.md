# web-m-resource

## 项目简介

`web-m-resource` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，所收录的页面均为独立、静态的 HTML 资源，用于展示、分享或存档用途。

## 目录说明

```
web-m-resource/
├── pages/          # 存放 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：按需组织 HTML 页面，每个页面独立存在，互不依赖。
- `assets/`：页面所需的公共或专用资源文件，建议按页面或功能子目录存放。

## 页面归档说明

- 所有页面均为独立的静态 HTML 文件，可直接在浏览器中打开。
- 页面内容可能包含演示、工具、信息展示或其他非商业用途的素材。
- 归档的页面不保证与任何线上服务同步，仅供本地或离线参考。

## 维护说明

- 本仓库由维护者不定期更新，欢迎提交 Pull Request 或 Issue 讨论。
- 新增页面请放置在 `pages/` 目录下，并避免引入外部依赖。
- 如需引用外部资源，建议下载至 `assets/` 目录后使用相对路径引用。
- 请勿在页面中加入恶意代码、跟踪脚本或侵犯他人权益的内容。

## 使用方式

1. 克隆或下载本仓库。
2. 使用浏览器打开 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后访问。
